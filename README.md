# orchestra
c string library

## Building
~~~sh
git clone https://github.com/asdfish/orchestra --depth=1
cd orchestra
make
~~~

## Usage
~~~sh
cc -Lorchestra -lorchestra -Iorchestra/include
~~~
If orchestra is cloned to a different directory, change -L and -I to the location of the repository.
